# ANYCABLE

This is the rails project with integrating the AnyCable.

* We have to install anycable-go server.
* It uses gRPC server.

## Procedure

Run Bundle install
`bundle install`

Run `rails g anycable:setup`

Run the server `rails s`.

Run `bundle exec anycable`

Run `~$ anycable-go`

Here you go!!!
